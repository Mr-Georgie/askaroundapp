# **AskAroundApp** 🌍✨  
**The hyperlocal Q&A app powered by AI**  
*Find the best spots near you—from barbers to burger joints—with answers from real locals.*  

---

## **🚀 Features**  
✅ **AI-Powered Answers** – Gemini summarizes crowdsourced replies into digestible insights.  
✅ **Anonymous Asking** – Post questions without revealing your identity.  
✅ **Manual Location Tags** – No Google Maps? No problem! Add your area as text.  
✅ **Real-Time Firebase Backend** – Questions/answers sync instantly.  
✅ **PWA Ready** – Installable on phones/desktops.  

---

## **🛠️ Tech Stack**  
- **Frontend**: Next.js + Tailwind CSS  
- **Backend**: Firebase (Firestore, Auth)  
- **AI**: Google GenKit + Gemini  
- **Styling**: Tailwind CSS + Heroicons  
- **Deployment**: Vercel

## **🚨 Why No Google Maps?**  
To keep costs $0 at launch, we use:  
- **Manual location input** (e.g., "Avenue Street").  
- **IP-based city detection** (via free APIs). - coming soon
*Planned upgrade: Map integration post-MVP!*  

---

## **🤖 How AI Works**    
1. **Answer Summaries**: AI compiles top-voted replies into one clean answer.  
2. **Question Matching**: Gemini finds duplicates using semantic search. - coming soon
3. **Spam Detection**: Flags promotional content automatically. - coming soon

## **🛠️ Setup**  
1. **Clone repo**:  
   ```bash  
   git clone https://github.com/mr-georgie/askaroundapp.git  
   cd askaroundapp  
   ```  
2. **Install dependencies**:  
   ```bash  
   npm install  
   ```  
3. **Configure Firebase**:  
   - Rename `.env.example` to `.env` and add your keys.  
4. **Run dev server**:  
   ```bash  
   npm run dev  
   ```
