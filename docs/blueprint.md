# **App Name**: AskAroundApp

## Core Features:

- Home Feed: Home feed displaying neighborhood questions in card format with voting, answer count, and location badges.
- Ask Question: Question form with auto-suggest feature to find similar questions. Users can type a question and the UI should display the available suggestions for similar question. The suggestions need not to come from an AI tool. For now it is enough that this comes from querying past data.
- Answer Question: Answer page with text input and optional photo upload functionality.
- Simulate Authentication: Simulate user authentication (no actual authentication mechanism will be set up in this MVP).
- Simulate Google Maps: Simulate Google Maps to display distances and locations.
- Ask Sage: Sage floating action button to ask an AI powered chatbot neighborhood related questions. It's a general purpose tool for helping you frame your question correctly.
- Push Notifications: Enable push notifications using Firebase to alert users of new answers to questions.

## Style Guidelines:

- Primary color: Vibrant teal (#008080) to promote a friendly and trustworthy feel within the community.
- Background color: Light teal (#E0F8F8) for a calming and accessible aesthetic.
- Accent color: Bright orange (#FFA500) to highlight interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for modern readability.
- Use emoji icons like ✂️ for barbershops to categorize recommendations.
- Employ rounded cards for questions to give a modern look and feel. Follow mobile first principles for responsiveness.
- Use subtle transitions across different pages.
